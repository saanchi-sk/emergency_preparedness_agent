An intelligent AI assistant that helps individuals and families prepare for and respond to emergencies. Built with agentic AI patterns, this system uses multi-step reasoning and tool selection to provide personalized emergency preparedness guidance.

**Features**

*FEMA Funding Assistance*
* Check eligibility requirements for disaster assistance
* Step-by-step application guidance
* Information on types of aid available
* Tips for successful claims

*Emergency Go-Bag Builder*
  * Comprehensive checklists for emergency supplies
  * Essential items, additional supplies, and special needs
  * Storage and maintenance recommendations
  * Family-specific considerations

*Emergency Shelter Locator*
* Find nearby emergency shelters
* Distance, capacity, and services information
* Contact information and availability
* What to bring guidance

**Agentic AI Capabilities**
Multi-step Reasoning: Transparent thought process
Tool Selection: Intelligently chooses appropriate resources
Context Awareness: Understands and adapts to user needs
Actionable Guidance: Provides practical, implementable advice
